# Go Event-Driven — Tranining Summary

**Provider:** Three Dots Labs (Miłosz Smółka & Robert Laszczak, creators of the [Watermill](https://github.com/ThreeDotsLabs/watermill) library)

**URL:** https://threedots.tech/event-driven/

## Overview

An online, hands-on training course teaching **Event-Driven Architecture (EDA) using Go (Golang)**. Targeted at experienced backend engineers who want to build scalable and resilient distributed systems.

## Course Structure

- **20 modules**, **94 exercises**
- **20+ hours** of coding
- Work in your own IDE
- Lifetime access + mentorship and community support

## Key Topics Covered

- Asynchronous communication patterns
- Message brokers: Redis, Kafka, PostgreSQL
- CQRS (Command Query Responsibility Segregation)
- Event sourcing
- Message ordering and consistency
- Fault-tolerant system design
- Observability (metrics and tracing)
